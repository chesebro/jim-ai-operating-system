# Opportunity Framing Framework

Use this framework when evaluating a new idea, venture, consulting opportunity, or product concept.

## Questions to answer

1. What is the core problem?
2. Who experiences this problem most acutely?
3. How is the problem solved today?
4. Why does this matter now?
5. What wedge could make this viable?
6. What assumptions must be true?
7. What is the best next test?

## Output structure

- Core problem
- Target user or buyer
- Existing alternatives
- Why now
- Wedge
- Risks and assumptions
- Next experiment
